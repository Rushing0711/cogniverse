# 第10章 集群

## 10.1 集群的目标

- **高可用（High Availability）**，是当一台服务器停止服务后，对于业务及用户毫无影响。
- **突破数据量限制**，一台服务器不能存储大量数据，需要多台分担，每个存储一部分，共同存储完整个集群数据。
- **数据备份容灾**，单点故障后，存储的数据仍然可以在别的地方拉起。
- **压力分担**，由于多个服务器都能完成各自一部分工作，所以尽量的避免了单点压力的存在。
