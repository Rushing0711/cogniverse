## 1.2 🎯 对话内命令（在 `hermes` 聊天界面中使用）

在聊天界面输入 `/help` 可以随时查看所有命令。

### 会话管理

| 命令             | 别名     | 作用                     |
| :--------------- | :------- | :----------------------- |
| `/new`           | `/reset` | 开始全新会话（清空历史） |
| `/clear`         | —        | 清屏并开始新会话         |
| `/retry`         | —        | 重新发送最后一条消息     |
| `/undo`          | —        | 删除最后一轮对话         |
| `/title [名称]`  | —        | 为当前会话命名           |
| `/history`       | —        | 查看对话历史             |
| `/save`          | —        | 保存当前会话             |
| `/resume [名称]` | —        | 恢复指定的会话           |

### 模型与配置

| 命令         | 别名  | 作用                 |
| :----------- | :---- | :------------------- |
| `/model`     | —     | 查看或切换当前模型   |
| `/config`    | —     | 显示当前配置         |
| `/verbose`   | —     | 切换工具进度显示级别 |
| `/statusbar` | `/sb` | 切换底部状态栏       |

- **临时切换模型**（仅本次会话有效）：

```bash
$ /model deepseek-chat
```

- **指定提供商切换模型**（当有多个同名模型时，或是自定义模型时）：

```bash
$ /model gemma4:26b --provider custom:ollama-gemma4:26b
```

- **永久切换默认模型**（保存到配置）：

```bash
$ /model deepseek-chat --global
```

- **永久切换默认模型**（保存到配置，当有多个同名模型时，或是自定义模型时）：

```bash
$ /model gemma4:26b --provider custom:ollama-gemma4:26b --global
```

### 工具与技能

| 命令      | 别名 | 作用                       |
| :-------- | :--- | :------------------------- |
| `/tools`  | —    | 管理工具（查看/启用/禁用） |
| `/skills` | —    | 搜索、安装和管理技能       |
| `/cron`   | —    | 管理定时任务               |

### 其他实用命令

| 命令                  | 作用                                            |
| :-------------------- | :---------------------------------------------- |
| `/voice on/off`       | 开启/关闭语音输入（`Ctrl+B` 录音）              |
| `/personality [名称]` | 切换人格（如 `kawaii`、`concise`、`technical`） |
| `/skin [名称]`        | 切换界面主题（如 `ares`、`mono`、`slate`）      |
| `/quit` 或 `/exit`    | 退出聊天                                        |

## 1.3 💻 终端命令（在 shell 中使用）

| 命令                          | 作用                                   |
| :---------------------------- | :------------------------------------- |
| `hermes`                      | 启动交互式聊天                         |
| `hermes chat -q "问题"`       | 单次查询（不进入交互界面，执行后退出） |
| `hermes setup`                | 运行配置向导                           |
| `hermes setup model`          | 单独配置模型提供商                     |
| `hermes model`                | 交互式选择模型                         |
| `hermes tools`                | 配置工具开关（交互式菜单）             |
| `hermes doctor`               | 健康检查，诊断配置问题                 |
| `hermes status`               | 查看所有组件状态                       |
| `hermes config edit`          | 打开配置文件                           |
| `hermes config show`          | 查看当前配置                           |
| `hermes config set KEY VALUE` | 设置配置项                             |
| `hermes gateway start`        | 启动消息网关（后台服务）               |
| `hermes gateway stop`         | 停止网关                               |
| `hermes gateway status`       | 查看网关状态                           |
| `hermes sessions list`        | 列出历史会话                           |
| `hermes sessions browse`      | 交互式浏览会话                         |
| `hermes cron list`            | 列出定时任务                           |
| `hermes update`               | 更新到最新版本                         |
| `hermes --version`            | 查看版本号                             |

## 1.4 ⌨️ 常用快捷键（在聊天界面中）

| 快捷键                            | 作用             |
| :-------------------------------- | :--------------- |
| `Enter`                           | 发送消息         |
| `Meta+Enter` / `Esc` 然后 `Enter` | 换行（多行输入） |
| `Ctrl+C`                          | 中断 AI 响应     |
| `Tab`                             | 自动补全斜杠命令 |
| `↑ / ↓`                           | 浏览输入历史     |
| `Ctrl+L`                          | 清屏             |
