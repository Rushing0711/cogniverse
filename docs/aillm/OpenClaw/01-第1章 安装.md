## 1.2 🦞 OpenClaw 常用命令速查表

### 📋 核心管理命令

| 命令                    | 作用                           |
| :---------------------- | :----------------------------- |
| `openclaw --version`    | 查看版本号                     |
| `openclaw status`       | 查看 Agent 和 Gateway 状态     |
| `openclaw status --all` | 完整状态（含配置、模型、频道） |
| `openclaw doctor`       | 诊断并自动修复常见问题         |
| `openclaw update`       | 更新 OpenClaw 到最新版本       |
| `openclaw uninstall`    | 卸载 OpenClaw                  |

### 🚪 Gateway 服务管理

| 命令                         | 作用                     |
| :--------------------------- | :----------------------- |
| `openclaw gateway start`     | 启动 Gateway 服务        |
| `openclaw gateway stop`      | 停止 Gateway 服务        |
| `openclaw gateway restart`   | 重启 Gateway 服务        |
| `openclaw gateway status`    | 查看服务状态             |
| `openclaw gateway install`   | 安装后台服务（开机自启） |
| `openclaw gateway uninstall` | 卸载后台服务             |
| `openclaw gateway logs`      | 查看 Gateway 日志        |

### 💬 TUI 交互界面

| 命令                            | 作用                     |
| :------------------------------ | :----------------------- |
| `openclaw tui`                  | 打开终端交互界面（推荐） |
| `openclaw tui --agent <名称>`   | 指定 Agent 进入 TUI      |
| `openclaw tui --session <名称>` | 指定会话进入 TUI         |

| 快捷键    | 作用         |
| :-------- | :----------- |
| `Ctrl+C`  | 退出 TUI     |
| `Ctrl+L`  | 清屏         |
| `/help`   | 查看帮助     |
| `/new`    | 开始新会话   |
| `/reset`  | 重置当前会话 |
| `/model`  | 切换模型     |
| `/status` | 查看状态     |

### ⚙️ 配置管理

| 命令                                | 作用             |

| `openclaw onboard`                  | 交互式配置向导   |

| `openclaw config set <key> <value>` | 设置配置项       |

## 💡 快速上手

```bash
# 1. 检查一切是否正常
openclaw status

# 2. 启动交互界面
openclaw tui

# 3. 如果需要配置频道或模型
openclaw configure

# 4. 查看 Gateway 状态
openclaw gateway status

# 5. 如果遇到问题
openclaw doctor
```
