# 第1章 环境准备

Tailwind CSS 通过扫描您所有的 HTML 文件、JavaScript 组件以及任何其他模板中的类名来工作，生成相应的样式，然后将它们写入一个静态的
CSS 文件。

它快速、灵活且可靠——无需运行时。

[在Vite中使用](https://tailwindcss.com/docs/installation/using-vite)

将 Tailwind CSS 作为 Vite 插件进行安装是与 Laravel、SvelteKit、React Router、Nuxt 和 SolidJS 等框架集成最无缝的方式。

## 1 创建项目

```bash
$ pnpm create vite@latest tailwind4-01-basic --template vue-ts
$ cd my-project
```

## 2 安装核心依赖

在v4中，`tailwindcss`包本身就是引擎，而`@tailwindcss/vite`则是与Vite集成的官方插件。

```bash
$ pnpm add -D tailwindcss @tailwindcss/vite
```

## 3 配置`vite.config.ts`

v4最大的变化之一就是**零配置**优先。你不再需要`tailwind.config.js`和`postcss.config.js`文件。所有配置都可以直接在Vite插件中晚餐。

```ts
import {defineConfig} from 'vite'
// @ts-ignore
import tailwindcss from "@tailwindcss/vite";

export default defineConfig({
    plugins: [tailwindcss()]
})
```

## 4 导入 Tailwind CSS

在您的 CSS 文件中添加 @import 语句来导入 Tailwind CSS。

```css
@import "tailwindcss";
```

## 5 运行项目

```bash
$ pnpm dev
```

## 6 编译发布

```bash
$ pnpm build
```
