# 第1章 环境准备

[在Vite中使用](https://www.getbootstrap.cn/docs/5.3/getting-started/vite/)

将 Tailwind CSS 作为 Vite 插件进行安装是与 Laravel、SvelteKit、React Router、Nuxt 和 SolidJS 等框架集成最无缝的方式。

## 1 创建项目

```bash
$ pnpm create vite@latest bootstrap5-01-basic --template vue
$ cd bootstrap5-01-basic
```

## 2 安装核心依赖

安装 Bootstrap。我们还需要安装 Popper，因为我们的下拉菜单、弹出框和工具提示的定位都依赖于它。如果您不打算使用这些组件，可以省略 Popper 的安装步骤

```bash
# 当时最新版本 "bootstrap": "^5.3.8"
$ pnpm i --save bootstrap @popperjs/core
```

## 3 安装其他依赖

除了 Vite 和 Bootstrap 之外，我们还需要另一个依赖项（Sass）来正确导入和打包 Bootstrap 的 CSS。

```bash
# 当时最新版 "sass": "^1.97.1"
$ pnpm i --save-dev sass
```

现在我们已经安装并设置好了所有必要的依赖项，我们可以开始创建项目文件并导入 Bootstrap 了。

### 4.3 style.css=>style.scss（重点：导入Bootstrap的样式）

**导入 Bootstrap 的 CSS。** 将以下代码添加到 `src/styles.scss` 中，以导入 Bootstrap 的所有源 Sass 代码。

```scss
// Import all of Bootstrap’s CSS // [!code focus:1]
@import "bootstrap/scss/bootstrap"; // [!code focus:1]
```

### 4.4 main.js（重点：导入Bootstrapd的脚本）

**接下来，我们需要加载 CSS 并导入 Bootstrap 的 JavaScript。** 将以下代码添加到 `src/main.js` 中，即可加载 CSS 并导入 Bootstrap 的所有 JS 文件。Popper 将通过 Bootstrap 自动导入。

```js
import { createApp } from 'vue'
// Import our custom CSS // [!code focus:4]
import './style.scss'
// Import all of Bootstrap’s JS
import * as bootstrap from 'bootstrap'
import App from './App.vue'

createApp(App).mount('#app')

```
